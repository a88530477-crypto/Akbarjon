## 🚨 2. Problem Statement (Why is this system needed?)

| Current Problem | Consequence |
| :--- | :--- |
| **📞📄 Manual Order Processing** (Phone calls & paper tickets) | ➡ 30% order error rate ❌ <br> ➡ Customer complaints during peak hours 😠 |
| **📦 No Real-Time Inventory Visibility** | ➡ Frequent stock-outs of popular items ⚠ <br> ➡ Inefficient purchasing decisions 💸 |
| **👤 No Customer Loyalty System** | ➡ No order history or personalized marketing 🎯 <br> ➡ Difficult customer retention 📉 |

## 📌 4. Scope of the System

### ✅ In-Scope

| Module | Features |
| :--- | :--- |
| **👤 Customer Portal** | 🔐 Registration/Login · 📋 Interactive Menu · 🛒 Customizable Cart · 💳 Online Payment · 🚚 Order Tracking |
| **👨‍🍳 Kitchen Dashboard** | 📥 Digital Order Queue · 🔄 Status Updates · ⚠ Low-Stock Alerts · 📊 Simple Analytics |
| **👨‍💼 Manager Admin Panel** | ✏ Menu Management · 📦 Inventory Tracking · 📈 Detailed Sales Reports · 👥 Customer Database View |

### ❌ Out-of-Scope

| Excluded Feature | Reason |
| :--- | :--- |
| 📱 Native Mobile Apps | Web-based platform only |
| 🚗 3rd-Party Delivery Integration | Focus on internal processes |
| 💼 Employee Payroll | HR functionality not required |
| 📹 Kitchen Video Streaming | Security & bandwidth concerns |
